# 備忘錄

## 將當前所在目錄（含子目錄）所有檔案複製到某一【目錄路徑】

### 基本語法

```powershell
Copy-Item -Path "來源路徑" -Destination "目標路徑" -Recurse
```

### 示範舉例

將當前目錄所有檔案（含子目錄），複製到【目錄路徑】： `~\AppData\Roaming\Rime`

1. 顯示當前目錄

```powershell
$ Get-Location
```

2. 複製當前目錄到指定位置

```powershell
$ Copy-Item -Path . -Destination "~\AppData\Roaming\Rime" -Recurse
```
