# 閩拚字典作業程序書

以下作業程序說明：如何自【台羅拚音】字典檔，轉換成【閩拚方案】字典檔。

## 檔案清單

- 轉換程式： convert_tlpa_to_bp_for_rime_dict.py
- 輸入檔案： 【預設】[ProjectRoot]\tools\tl_ji_khoo_peh_ue.dict.yaml
- 輸出檔案： 【預設】[ProjectRoot]\bp_ji_khoo.dict.yaml

## 作業程序

### 預設模式

```bash
.\.venv\Scripts\Activate.ps1
py tools\convert_tlpa_to_bp_for_rime_dict.py
```

### 自訂模式

```bash
.\.venv\Scripts\Activate.ps1
py tools\convert_tlpa_to_bp_for_rime_dict.py tools\ji_khoo_II.dict.yaml tools\bp_ji_khoo_new.dict.yaml
```
